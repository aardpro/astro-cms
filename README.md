# astro-cms
> A SEO-friendly, Astro-based responsive blog/CMS system with optimized content management.    
> 一个基于 Astro 开发的 SEO 友好、响应式博客/内容管理系统。    

## 部署
***注意拉取docker需要魔法上网***
```bash
git clone https://github.com/aardpro/astro-cms.git
cd astro-cms
docker compose up -d
# 然后通过 http://127.0.0.1:3000访问
# 管理员账号：admin
# 管理员初始密码：admin
```
